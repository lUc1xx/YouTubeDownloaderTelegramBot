# Getting Started

First, install packages

```bash
npm install
# or
yarn
```

After, start development

```bash
npm run dev
# or
yarn dev
```

# Usage

For build application 

```bash
npm run dev
# or
yarn dev
```

For start application 

```bash
npm run start
# or
yarn start
```

Make sure to include your bot credentials in the .env file or as environment variables.
